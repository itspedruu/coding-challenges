# Problem

Given a function `f(x, y)` and a value `z`, return all positive integer pairs `x` and `y` where `f(x,y) == z`.

# What we know

* `f(x, y) < f(x + 1, y)`
* `f(x, y) < f(x, y + 1)`
* `1 <= x, y <= 1000`
